# TenThousand

A minimalist macOS menubar app for tracking time spent on skills. Built with SwiftUI, it helps you work toward mastery by logging practice hours with persistent local storage.

## Features

- **Skill Tracking** - Create and manage multiple skills with color-coded organization
- **Time Sessions** - Start, pause, resume, and stop tracking with accurate timing
- **Activity Heatmaps** - Visual overview of practice activity over time
- **Keyboard Shortcuts** - Fast navigation with `⌘N` (new skill), `⌘Q` (quit), arrow keys
- **Local Storage** - All data stored privately on your Mac with CoreData
